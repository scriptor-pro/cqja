# agent.md — CQJA (Ce Que J’ai Appris)

## Vision
CQJA est un objet éditorial structuré, minimal et intellectuel léger.
Publication terminal-first via Eleventy + Cloudflare Pages.

## Stack
- Eleventy
- Node LTS
- Cloudflare Pages
- Wrangler CLI

## Workflow
note -t tags "Titre"
npm run build
publish

## Fonctionnalités V1
- Numérotation automatique
- Navigation précédente/suivante
- Nuage de tags pondéré + filtre JS
- Liens externes sécurisés (_blank + noopener)
- Accessibilité WCAG 2.1 AA

## V2 (non implémenté)
- Manifeste
- Mode sombre
- Export PDF
